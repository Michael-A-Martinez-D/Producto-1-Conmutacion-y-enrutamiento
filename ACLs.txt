ip access-list extended Restriccion-Estudiantes
remark permitir DHCP
permit udp any eq bootpc any eq bootps
remark evaluar si hay sesiones de gestion 
evaluate SESIONES_GES
remark Permitir HTTP y HTTPS hacia el servidor 
permit tcp 172.18.0.0 0.0.15.255 host 172.18.18.10 eq www
permit tcp 172.18.0.0 0.0.15.255 host 172.18.18.10 eq 443
remark Denegar cualquier otro trafico hacia el servidor 
deny ip 172.18.0.0 0.0.15.255 host 172.18.18.10
remark denegar trafico hacia las demas subredes
deny ip 172.18.0.0 0.0.15.255 172.18.17.128 0.0.0.127
deny ip 172.18.0.0 0.0.15.255 172.18.17.0 0.0.0.127
deny ip 172.18.0.0 0.0.15.255 172.18.18.64 0.0.0.31
deny ip 172.18.0.0 0.0.15.255 172.18.16.0 0.0.0.255
remark permitir todo lo demas (salida a internet)
permit ip 172.18.0.0 0.0.15.255 any


interface vlan 10
ip access-group Restriccion-Estudiantes in

ip access-list extended Restriccion-Docentes
remark permitir DHCP
permit udp any eq bootpc any eq bootps
remark evaluar si hay sesiones de gestion

deny ip 172.18.16.0 0.0.0.255 172.18.17.128 0.0.0.127
deny ip 172.18.16.0 0.0.0.255 172.18.18.64 0.0.0.31
remark permitir todo lo demas

interface vlan 20

ip access-list extended ACL_GES
Remark Rastrear tráfico TCP hacia Estudiantes
permit tcp 172.18.18.64 0.0.0.31 172.18.0.0 0.0.15.255 reflect SESIONES_GES
Remark Rastrear tráfico UDP hacia Estudiantes
permit udp 172.18.18.64 0.0.0.31 172.18.0.0 0.0.15.255 reflect SESIONES_GES
Remark Rastrear tráfico ICMP hacia Estudiantes
permit icmp 172.18.18.64 0.0.0.31 172.18.0.0 0.0.15.255 reflect SESIONES_GES
 
Remark Rastrear tráfico TCP hacia Docentes
permit tcp 172.18.18.64 0.0.0.31 172.18.16.0 0.0.0.255 reflect SESIONES_GES
Remark Rastrear tráfico UDP hacia Docentes
permit udp 172.18.18.64 0.0.0.31 172.18.16.0 0.0.0.255 reflect SESIONES_GES
Remark Rastrear tráfico ICMP hacia Docentes
permit icmp 172.18.18.64 0.0.0.31 172.18.16.0 0.0.0.255 reflect SESIONES_GES
 
Remark Permitir el resto del tráfico normal de Gestión hacia cualquier otro lado
permit ip 172.18.18.64 0.0.0.31 any
 
int vlan 30
ip access-group ACL_GES in
